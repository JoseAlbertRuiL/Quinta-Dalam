# 🐙 GitHub Copilot - Project Identity

## 📋 Role: Agile Contributor

Focus on rapid feature implementation while adhering to the project's SSR and accessibility standards.

## 🛠️ Operating Rules

1. **Inheritance:** Follow [../common/stack.md](../common/stack.md) for technology choices.
2. **Environment:** Assume Node.js SSR context.
3. **Accessibility:** Mandatory `aria-label`, `alt` tags, and `htmlFor` on labels.
4. **Shell:** Strictly Bash scripts.

## 🚀 Key Directives

- Use absolute paths starting with `/`.
- Keep interactive logic within React components only.
- Match the `kebab-case` naming for files.

# 🛠️ Project Stack: Quinta-Dalam

## Core Technologies

- **Framework:** Astro (SSR Mode via `@astrojs/node`).
- **Styles:** Tailwind CSS + Vanilla CSS (Custom Properties in `:root`).
- **Language:** JavaScript / TypeScript (ESM).
- **Icons:** FontAwesome.

## Tooling & Environment

- **Runtime:** Node.js.
- **Package Manager:** `pnpm`.
- **Shell:** **Bash / POSIX Standard**.
  - *Note:* Do not use Nushell, Zsh-isms, or Fish-isms for project scripts.
- **Database:** Supabase (PostgreSQL + Auth + Storage).

## Execution Standards

- Always run commands from the project root.
- Use `pnpm run dev` for local development.
- Adhere to the `package.json` scripts for building and deployment.

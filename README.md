## 🌟 Propósito
El **Hotel Quinta Dalam** es una propuesta web moderna y responsiva diseñada para capturar la esencia cultural de los Pueblos Mágicos de Michoacán. Este proyecto busca proporcionar una experiencia de usuario fluida, siguiendo los estándares actuales de desarrollo (HTML5, CSS3, JS), cumpliendo con los requerimientos académicos de la materia de **Programación Web** así como un futuro espacio de desarrollo para un entorno profesional o laboral.

---

## 🚀 Características Principales
- **🎨 Experiencia Temática**: 15 habitaciones inspiradas en la riqueza de Michoacán.
- **📱 Diseño Responsive**: Optimizado para dispositivos móviles y escritorio. [Ver detalles en el Roadmap](#roadmap-de-implementación)
- **💬 Interacción Directa**: Integración con redes sociales y reserva vía WhatsApp.
- **📽️ Multimedia**: Incorporación de showcase en video para previsualizar la experiencia.

## 📂 Estructura del Proyecto
```bash
Hotel_PaginaWeb/
├── 📂 css/           # Estilos (Reset, Styles, Responsive, Grid, Flexbox)
├── 📂 images/        # Banco de imágenes y recursos gráficos
├── 📂 media/         # Videos y contenido multimedia
├── 📂 pages/         # Subpáginas del ecosistema
│   ├── acerca-de.html      # Acerca del Hotel y sus inicios, misión, visión, etc.
│   ├── habitaciones.html   # Información acerca de las habitaciones y su cultura
│   ├── instalaciones.html  # Showcase de las instalaciones generales del Hotel
│   ├── privacidad.html     # Política de Privacidad
│   ├── reservaciones.html  # Módulo de Reservaciones tipo Forms
│   ├── servicios.html      # Servicios disponibles
│   ├── tours.html          # Tours del Hotel - Posible Eliminación
│   └── tyc.html            # Términos y Condiciones
├── 📂 admin/         # Panel de Administración
│   ├── admin.html          # Inicio de sesión para administradores
│   ├── balance.html        # Finanzas, balance y caja del Hotel
│   ├── database.html       # Gestión de base de datos
│   ├── dev.html            # Herramientas de desarrollo
│   ├── inbox.html          # Mensajes y Correos pendientes por leer
│   ├── panel-admin.html    # Dashboard principal del administrador
│   └── stats.html          # Estadísticas y analíticas del hotel
└── index.html        # Página Principal
```

## 💻 Entorno de Desarrollo y Convenciones

### Herramientas
- **IDE**: Visual Studio Code / Antigravity
- **Formateador**: **Prettier** para mantener indentación y claridad impecables.
- **Visualización**: **Live Preview** para previsualización local en tiempo real.

### 📝 Convenciones de Comentarios (Better Comments)
Utilizamos colores específicos para facilitar la lectura técnica tanto para desarrolladores como evaluadores:

*   🔴 `!` **Crítico**: Partes esenciales del núcleo. No modificar sin conocimiento previo.
*   🔵 `*` **Informativo**: Documentación de secciones, imágenes o bloques relevantes.
*   🟢 `?` **Nota**: Detalles específicos sobre funcionalidades curiosas o minimalistas.
*   🟠 `TODO` **Pendiente**: Mejoras, arreglos o cambios futuros planificados.
